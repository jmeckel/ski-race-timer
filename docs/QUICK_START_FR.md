# Ski Race Timer - Démarrage rapide

**Opérationnel en moins de 2 minutes.**

Version 5.26.2

---

## Chronométreur

1. **Installer l'application** -- Ouvrez l'URL dans votre navigateur. iOS : Partager > Sur l'écran d'accueil. Android/Chrome : Menu > Installer l'application.
2. **Compléter l'assistant** -- Choisissez votre langue, définissez un nom d'appareil (ex. « Départ »), sélectionnez le rôle Chronométreur, et activez éventuellement la capture photo et la synchronisation cloud.
3. **Saisir le numéro de dossard** -- Appuyez sur les chiffres du cadran rotatif pour entrer le dossard du coureur (ex. appuyez sur 5 pour le dossard 005).
4. **Sélectionner le point de chronométrage et la manche** -- Appuyez sur **D** (Départ) ou **A** (Arrivée) à gauche du centre du cadran. Appuyez sur **M1** ou **M2** à droite pour le numéro de manche.
5. **Enregistrer le temps** -- Appuyez sur le grand bouton **Enregistrer le temps** au moment où le coureur franchit la ligne. L'horodatage synchronisé par GPS est sauvegardé automatiquement.

---

## Juge de porte

1. **Sélectionner le rôle Juge de porte** -- Dans l'assistant de configuration (ou dans les Paramètres), choisissez Juge de porte comme rôle de l'appareil.
2. **Affecter vos portes** -- Entrez la plage de portes dont vous êtes responsable (ex. portes 5-12) et sélectionnez la manche (M1/M2).
3. **Appuyer sur la porte** -- Lorsque vous observez une faute, appuyez sur le numéro de porte dans la grille. Le panneau de détail de la faute s'ouvre en dessous.
4. **Sélectionner le type de faute** -- Choisissez **PM** (porte manquée), **ENF** (enfourché) ou **FO** (fixation ouverte). Le dossard du coureur en piste se remplit automatiquement.
5. **Enregistrer** -- Appuyez sur **Enregistrer la faute**. La porte et le dossard restent sélectionnés pour enregistrer rapidement les fautes suivantes.

---

## Directeur de course

1. **Activer la synchronisation cloud** -- Dans les Paramètres, entrez un ID de course et un PIN à 4 chiffres pour que tous les appareils partagent les mêmes données.
2. **Accéder au mode Directeur de course** -- Depuis l'onglet Résultats, ouvrez le panneau Directeur de course. Le PIN vous sera demandé pour vérification.
3. **Examiner les fautes** -- Parcourez les fautes soumises par les juges de porte. Approuvez ou rejetez les demandes de suppression de fautes.
4. **Définir le mode de pénalité** -- Choisissez **PEN** (pénalité de temps par faute, pour les catégories U8/U10) ou **DSQ** (disqualification) selon le règlement de la course.
5. **Exporter les résultats** -- Depuis l'onglet Résultats, exportez en CSV au format Race Horology pour le traitement officiel de la course.

---

## Configuration multi-appareils

1. **Utilisez le même ID de course** sur chaque appareil (chrono départ, chrono arrivée, juges de porte).
2. **Utilisez le même PIN à 4 chiffres** sur chaque appareil pour authentifier la synchronisation cloud.
3. **Donnez un nom unique à chaque appareil** (ex. « Départ », « Arrivée », « Porte 5-12 ») pour identifier les entrées.

Tous les appareils se synchronisent automatiquement toutes les 5 secondes lorsqu'ils sont connectés.
