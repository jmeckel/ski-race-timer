01

Introduction

Ski Race Timer is a GPS-synchronized Progressive Web App (PWA) for timing alpine ski races. Designed for mobile use in outdoor race conditions, it provides accurate, coordinated timing across multiple devices.

Key Features

  • GPS-synchronized timing for accurate timestamps across devices
  • Multi-device sync to coordinate start and finish timers
  • Gate Judge mode with gate-first quick entry for recording gate faults
  • Chief Judge mode for fault review, penalty decisions, and deletion approvals
  • Two-run support for slalom and giant slalom races
  • Voice Mode for hands-free voice commands during timing
  • Ambient Mode to save battery by dimming the screen during inactivity
  • Battery Power Saver — automatic animation reduction on low battery
  • FLT status for U8/U10 categories (finished with fault penalty)
  • Offline-first — works without internet, syncs when connected
  • Photo capture — optional documentation for each timestamp
  • Race Horology export — industry-standard CSV format
  • Trilingual — English, German, and French interfaces

Einleitung

Ski Race Timer ist eine GPS-synchronisierte Progressive Web App (PWA) zur Zeitmessung bei alpinen Skirennen. Für den mobilen Einsatz bei Rennen im Freien konzipiert, bietet sie eine präzise, geräteübergreifende Zeitmessung.

Introduction

Ski Race Timer est une application web progressive (PWA) synchronisée par GPS pour le chronométrage des courses de ski alpin. Conçue pour une utilisation mobile en conditions extérieures, elle fournit un chronométrage précis et coordonné sur plusieurs appareils.

Fonctionnalités principales

  • Chronométrage synchronisé par GPS pour des horodatages précis sur tous les appareils
  • Synchronisation multi-appareils pour coordonner les chronométrages
  • Mode Juge de porte avec saisie rapide porte-en-premier
  • Mode Directeur de course pour la revue des fautes et décisions de pénalité
  • Commande vocale pour utilisation mains libres sur la piste
  • Mode veille pour protéger contre les horodatages accidentels
  • Économiseur de batterie — réduction automatique des animations
  • Statut FLT pour les catégories U8/U10
  • Support deux manches pour le slalom et le slalom géant
  • Hors ligne d'abord — fonctionne sans internet
  • Capture photo — documentation optionnelle
  • Export Race Horology — format CSV standard
  • Trilingue — interfaces en allemand, anglais et français

03

Timer View

The main workspace features a radial dial (iPod-style) for quick bib number input along with timing controls.

Radial dial timer view

Recording Times

  1. Enter bib number using radial dial or number keys (displays as 3 digits)
  2. Select timing point — Start (S) or Finish (F)
  3. Select run — L1 or L2
  4. Tap “Record Time”

Radial Dial

Digits 0–9 are arranged in a circle. Tap to enter a digit or use the spin gesture.

Spin Gesture

  • Clockwise = increment
  • Counter-clockwise = decrement
  • Momentum physics for fast spinning
  • Snap-back after 800ms of inactivity

Timing Points & Run Selection

Toggle between Start (S) and Finish (F) timing points. Select L1 (first run) or L2 (second run) for slalom races.

Additional Features

  • Auto-Increment — bib number increases by 1 after recording Finish
  • Duplicate Warning — yellow warning for same bib+point+run
  • Landscape Mode — two-column layout (dial left, controls right)
  • Wake Lock — screen stays active during timing
  • Ambient Mode — dims after 30s, first tap wakes, second tap records

Timer-Ansicht

Der Hauptarbeitsbereich mit radialem Zifferblatt (iPod-Stil) zur schnellen Startnummern-Eingabe und Zeitmesssteuerung.

Radiales Zifferblatt

Zeiten erfassen

  1. Startnummer eingeben (0–9 auf Zifferblatt, 3-stellig)
  2. Messpunkt: S (Start) oder Z (Ziel)
  3. Lauf: L1 oder L2
  4. „Zeit erfassen“ tippen

Radiales Zifferblatt

Ziffern 0–9 im Kreis angeordnet. Antippen oder Drehen zur Eingabe.

Drehgeste

  • Im Uhrzeigersinn = erhöhen
  • Gegen den Uhrzeigersinn = verringern
  • Schwungphysik für schnelles Drehen
  • Rückschnappen nach 800ms Inaktivität

Messpunkte & Laufauswahl

Umschalten zwischen Start (S) und Ziel (Z). Laufauswahl L1 oder L2 für Slalom-Rennen.

Weitere Funktionen

  • Auto-Inkrement — Startnummer +1 nach Ziel-Erfassung
  • Duplikat-Warnung — gelbe Warnung bei gleicher Kombination
  • Querformat — Zwei-Spalten-Layout
  • Bildschirmsperre — Bildschirm bleibt aktiv
  • Ruhemodus — dimmt nach 30s, erstes Tippen weckt, zweites erfasst

Vue chronomètre

L'espace de travail principal avec cadran radial (style iPod) pour la saisie rapide des dossards et contrôles de chronométrage.

Vue cadran radial

Enregistrer des temps

  1. Saisir le dossard (cadran ou touches, 3 chiffres)
  2. Point de mesure : Départ (S) ou Arrivée (F)
  3. Manche : M1 ou M2
  4. Appuyer sur « Enregistrer le temps »

Cadran radial

Chiffres 0–9 disposés en cercle. Appui direct ou rotation pour la saisie.

Geste de rotation

  • Horaire = incrémenter
  • Antihoraire = décrémenter
  • Physique d'inertie pour rotation rapide
  • Retour après 800ms d'inactivité

Points de mesure & sélection de manche

Basculer entre Départ (S) et Arrivée (F). Sélectionner M1 ou M2 pour les courses de slalom.

Fonctionnalités supplémentaires

  • Auto-incrémentation — dossard +1 après arrivée
  • Avertissement de doublon — avertissement jaune
  • Mode paysage — deux colonnes
  • Verrouillage écran — écran actif
  • Mode veille — atténue après 30s, première pression réveille, deuxième enregistre

Gate Judge Mode

Gate Judge view

Gate-First Workflow

  1. Tap a gate number in the grid
  2. Bib number auto-fills from the current racer
  3. Select fault type
  4. Save the fault entry
Minimum 2 taps

The gate-first workflow is designed for speed: tap gate, then confirm — a minimum of 2 taps to record a fault.

Setup

Settings → Gate Judge role → Set gate range → Select run

Fault Types

CodeType
MGMissed Gate
STRStraddled
BRBinding Release

Gate Grid

  • 5-column layout
  • Red = odd gates, Blue = even gates
  • Fault count badges on each gate

Voice Notes

After recording a fault, you can add a voice note that is transcribed in real-time (max 500 characters).

Ready Status

Signal readiness for the next racer to coordinate with other judges and the chief judge.

Layout

Header → Gate Grid → Fault Detail → Recorded Faults → Footer (Save + Ready)

Troubleshooting

Sync Issues

  • Check that the Race ID matches on all devices
  • Verify the PIN is correct
  • Pull-to-refresh to trigger a manual sync
  • Toggle sync off and on again

GPS Issues

  • Enable GPS/Location Services on your device
  • Grant location permission to the browser
  • Use the app outdoors with a clear sky view
  • Wait 30–60 seconds for the initial GPS fix

Camera Issues

  • Grant camera permission to the browser
  • Enable photo capture in Settings
  • Reload the app if the camera does not start

Wake Lock Issues

  • Enable Ambient Mode to keep the screen active
  • Adjust your device’s screen timeout settings

Voice Mode Issues

  • Voice Mode requires an internet connection
  • Grant microphone permission to the browser
  • Speak clearly and avoid background noise

Ambient Mode Issues

By Design

The first tap on the dimmed screen only wakes the display. The second tap records the timestamp. This is intentional to prevent accidental recordings.

General

  • Clear the browser cache and reload
  • Reinstall the PWA from the browser
  • Try a different browser (Chrome, Safari, Firefox)

Fehlerbehebung

Sync-Probleme

  • Rennen-ID auf allen Geräten prüfen
  • PIN überprüfen
  • Pull-to-Refresh für manuelle Synchronisation
  • Sync aus- und wieder einschalten

GPS-Probleme

  • GPS/Standortdienste auf dem Gerät aktivieren
  • Standort-Berechtigung im Browser erteilen
  • App im Freien mit freier Sicht verwenden
  • 30–60 Sekunden auf GPS-Fix warten

Kamera-Probleme

  • Kamera-Berechtigung im Browser erteilen
  • Fotoaufnahme in Einstellungen aktivieren
  • App neu laden, falls Kamera nicht startet

Bildschirmsperre-Probleme

  • Ruhemodus nutzen, um Bildschirm aktiv zu halten
  • Geräte-Timeout in den Systemeinstellungen anpassen

Sprachsteuerung-Probleme

  • Internetverbindung erforderlich
  • Mikrofon-Berechtigung im Browser erteilen
  • Deutlich und ohne Hintergrundgeräusche sprechen

Ruhemodus-Probleme

Beabsichtigt

Das erste Tippen auf den gedimmten Bildschirm weckt nur die Anzeige. Das zweite Tippen erfasst den Zeitstempel. Dies ist absichtlich so, um versehentliche Erfassungen zu vermeiden.

Allgemein

  • Browser-Cache leeren und neu laden
  • PWA über den Browser neu installieren
  • Anderen Browser versuchen (Chrome, Safari, Firefox)

Dépannage

Problèmes de synchronisation

  • Vérifier l'identifiant de course sur tous les appareils
  • Vérifier le PIN
  • Tirer pour actualiser (synchronisation manuelle)
  • Désactiver et réactiver la synchronisation

Problèmes GPS

  • Activer le GPS/Services de localisation
  • Accorder l'autorisation de localisation au navigateur
  • Utiliser l'application en extérieur avec ciel dégagé
  • Attendre 30–60 secondes pour le premier fix GPS

Problèmes de caméra

  • Accorder l'autorisation caméra au navigateur
  • Activer la capture photo dans les paramètres
  • Recharger l'application si la caméra ne démarre pas

Problèmes de verrouillage écran

  • Activer le mode veille pour garder l'écran actif
  • Ajuster le délai d'écran de l'appareil

Problèmes de commande vocale

  • Connexion internet requise
  • Accorder l'autorisation du micro au navigateur
  • Parler clairement et éviter le bruit de fond

Problèmes du mode veille

Intentionnel

La première pression sur l'écran atténué ne fait que réveiller l'affichage. La deuxième pression enregistre l'horodatage. C'est intentionnel pour éviter les enregistrements accidentels.

Général

  • Vider le cache du navigateur et recharger
  • Réinstaller la PWA depuis le navigateur
  • Essayer un autre navigateur (Chrome, Safari, Firefox)
